# lexispawn skills

the crowd reads the surface. these read the current.

One record, one forge, five words-skills. Every skill is a single `SKILL.md`, portable, no keys, no dependencies. Drop it into an agent that reads skills, or read it yourself.

## the record

- **[riptide](riptide_market_read/)** reads prediction and probability markets. Name the surface, find the current beneath it, call the edge numberless and honest. The original council; the live record runs at [lexispawn.xyz/predictions](https://lexispawn.xyz/predictions).

## the forge

- **[lex-spawn](lex-spawn/)** turns any capability or repeated workflow into a complete, catalog-grade skill: five gates from seed to ship, collision-swept, secret-swept, born with its own eval manifest. Agents that only install skills stay the size of the catalog. Agents that mint them compound.

## the lexicon suite

every token, agent, and app ships with words on day one. these make the words carry.

- **[lex-memetic-density](lex-memetic-density/)** writes what a thing feels like, compresses until it survives retelling, strips hedges.
- **[lex-token-lexicon](lex-token-lexicon/)** gives a token its own language: seed words, holder name, event names, the drop-line.
- **[lex-true-name](lex-true-name/)** names things so they spread: mouth-feel, meaning stack, live collision sweep, sleep on the shortlist.
- **[lex-voice-audit](lex-voice-audit/)** catches the drift that turns an account into every other bot. scorecard, banned list, rewrites.
- **[lex-fee-loop](lex-fee-loop/)** runs the creator loop on purpose: content, mindshare, volume, fees, reinvest. weekly, with numbers.

## install

into a bankr agent:

```
install the lex-spawn skill from https://github.com/BankrBot/skills/tree/main/lex-spawn
```

same shape for the others. or take any `SKILL.md` straight from this repo into any agent that reads the format.
