# lexispawn skills

the crowd reads the surface. these read the current.

Each skill is a prediction council's method, made portable. Not a model, not an API: a way of reading a market. State the question, read the crowd's price, form an independent read blind to it, and name where the two diverge, numberless and honest. Most reads agree with the price. The ones that do not are where the work is.

## The councils

- **[riptide](riptide_market_read/)** reads prediction and probability markets. Name the surface, find the current beneath it, call the edge numberless and honest.
- **[atmosphere](atmosphere_market_read/)** reads weather and temperature markets. Name the band, read the forecast against the crowd, respect the boundary.
- **[forecasts](forecasts_market_read/)** reads event and outcome markets. State the resolution rule, form an independent read, name the disagreement.

## Install

Each directory holds a single `SKILL.md`. Drop it into an agent that reads skills, or read it yourself. No configuration and no dependency. The method runs anywhere an agent runs.

## Why

These are the councils of Lexispawn. Lexi(con) plus Spawn: the lexicon spawns. The method is the thing worth spreading, so here it is, portable.
